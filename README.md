# The-Bollywood-Game
This is a Bollywood game in which a Bollywood movie will be chosen at random from the list of movies and will be presented in the form of "*".The player has to guess a movie by using hints available The Rules: 1)The player will get a limited number chances and he has to guess the correct movie within those attempts available 2)The player can guess a letter or complete movie name at once but if his choice of letter is not present or the movie name he guessed is incorrect his number of number of attempts will decrease by one. 3)The player can also ask for help by typing "hint" in the solution . 4)There are in total 2 hints available for the user per movie. 5)If the player guess the movie he wins and the program terminates 6)if layer runs out of number of attempts the player loses
